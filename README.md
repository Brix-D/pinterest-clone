***

PINTEREST CLONE

****

Это попытка создать приближенный проект к популярному хостингу изображений.

Стек: Golang 1.25, Gorm, Meilisearch, S3 хранилище
Фронтенд: vue 3, nuxt 4, ui kit: PrimeVue или Nuxt/UI, Tailwind, кое-где scss и css modules

Задачи:
1. авторизация через OAuth,
2. хранение artworks - в базе данных, самих media-файлов в S3
3. каждый пост и artwork будет иметь теги один или много
4. поиск будет производится полнотекстово по публичным artworks, в поиске будет режим "выключить NSWF" (безопасный поиск)
5. будет лента рекомендаций, созданная без сложных алгоритмов, просто самое последнее загруженное
6. будет личный кабинет художника: загрузка работ, редактирование тегов, статистика по просмотрам
7. личный кабинет пользователя - сохраненые работы
